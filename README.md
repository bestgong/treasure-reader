# 🗡️ 寻宝小英雄 · 环球阅读冒险

一个为 8 岁孩子打造的 **《环球寻宝记》系列纸质书伴读平台**。每读完一本书，孩子就能在网页上完成对应国家的互动答题、收集国宝、盖章护照。

## ✨ 特点

- 🗺️ **世界地图首页**：直观展示已解锁/待解锁国家
- 📔 **护照印章系统**：读完一本盖一章，跨书累积成就感
- 🏛️ **国宝收集册**：日本七支刀、埃及面具、希腊金瓶…集齐一件解锁一件
- 👥 **角色图鉴**：布卡、麦克、峰巴巴等系列角色
- 🎯 **每章 4-5 道细节问题**：考察孩子是否读得仔细
- 💡 **读前钩子 / 读中互动 / 读后任务 / 小百科** 四段式陪读
- 💾 **本地自动保存**：localStorage 永不丢失进度
- 🌸 **童趣可爱风格**：樱花粉 + 朱红 + Q 版 emoji

## 📚 当前已上线

```
treasure-reader/
├── index.html              # 主入口
├── src/
│   ├── app.js              # 主应用逻辑
│   ├── style.css           # 童趣可爱样式
│   └── data/
│       ├── countries.js    # 国家注册表（地图坐标、角色图鉴）
│       └── japan.js        # 日本寻宝记 10 章数据

```

## ➕ 如何新增一本书

1. 在 `src/data/` 新建国家文件（参考 `japan.js`）
2. 在 `src/data/countries.js` 中 `import` 并注册
3. 把 `worldMap` 中对应国家的 `available` 改成 `true`
4. 提交并 push，GitHub Pages 自动更新

## 💝 给爸爸妈妈的话

这个网页**不是替代书**，而是让孩子读书更有动力、更仔细。请配合纸质书使用：
- 让孩子先读纸质书的一章
- 再来网页做对应章节的"读完挑战"
- 答错没关系，让 ta 翻回书里找答案——这就是细读的过程
